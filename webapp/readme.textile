h1. Semantic Feedback

A web service for capturing online feedback, that will automatically analyse the text content, categorize it and qualify it.

By "Sérgio Santos":http://sergiosantos.info and "Marco Simões":http://twitter.com/disbeat


h2. Requirements

* simplejson
* "wikipydia":http://github.com/j2labs/wikipydia
